How to play

Guess the secret five-letter word in six tries.

  • Green means the letter is in the correct spot.
  • Gold means the letter is in the word but in a different spot.
  • Slate means the letter is not in the word.
  • Set the current player so wins and losses go onto the family scoreboard.

The game supports touch, mouse, and hardware keyboard input.
